Netflix Recommender — Página de error

Hemos preparado páginas de error estilizadas para 403, 502 y 503. Usa el selector para previsualizar cada una.

403
Acceso denegado

No tienes permiso para ver este recurso.

Posible causaCredenciales insuficientes o política de acceso
Qué hacerInicia sesión con una cuenta autorizada o solicita acceso al administrador.

Si crees que esto es un error, intenta cerrar sesión e iniciar de nuevo, o contacta con el equipo de soporte indicando el recurso al que intentabas acceder.

RecargarIr a inicio
502
Bad Gateway

El servidor recibió una respuesta inválida desde el upstream.

Posible causaProblema temporal con el servidor remoto o balanceador.
Qué hacerEspera unos segundos e inténtalo de nuevo. Si persiste, revisa logs y configuración de proxy.

Este error suele aparecer cuando un proxy o gateway no puede conectarse o recibe respuesta inválida del servidor de origen.

ReintentarVer sugerencias
503
Servicio no disponible

El servicio está temporalmente fuera de servicio.

Posible causaMantenimiento, sobrecarga o despliegue en curso.
Qué hacerComprueba ventanas de mantenimiento, autoscaling y límites de recursos.

Si estás realizando un despliegue o mantenimiento, este estado es esperado. Si no, revisa métricas y colas de trabajo para identificar cuellos de botella.

Consejos rápidos

Logs y diagnósticos

Revisa logs del servidor, del proxy y de la plataforma. Para 502/503, mira timeouts, probes de salud y límites de recursos. Para 403, revisa RBAC y reglas de firewall.

Pruebas

Usa curl para reproducir la petición y ver encabezados: curl -I https://tusitio/endpoint. Revisa los headers X-Forwarded-For y cualquier header de proxy.
